## Features

- AI powered smart recipient matching
- Food freshness prediction
- Live map with route optimization
- Confetti celebration popup when food is accepted
- Daily updates feed
- SDG goals tracker
- Weekly impact chart
- Smart fallback system

## Installation

### Backend
git clone https://github.com/shivasumedha/FoodBridge-AI-.git
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
cd backend
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm start
